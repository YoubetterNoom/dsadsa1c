User Guide ×

Basic Operations

  • Select Image: Click on any image from the sidebar to add it to the canvas.
  • Move Image: Click and drag an image to move it around the canvas.
  • Deselect: Click on empty space on the canvas to deselect any selected image.

Image Manipulation

  • Resize: Drag the corner handles to resize the selected image.
  • Rotate: Drag the rotate handle (circle above the image) to rotate.
  • Flip Horizontal: Mirror the selected image horizontally (left-right).
  • Flip Vertical: Mirror the selected image vertically (up-down).

Layer Controls

  • Bring Forward: Move the selected image one layer up.
  • Send Backward: Move the selected image one layer down.
  • Bring to Front: Move the selected image to the top layer.
  • Send to Back: Move the selected image to the bottom layer.

Text Functions

  • Add Text: Click the "Add Text" button to add text to the canvas.
  • Edit Text: Double-click on any text element to edit its content.
  • Format Text: Use the text controls that appear when text is selected.

Other Functions

  • Delete Image: Select an image and press the Delete key to remove it from the canvas.
  • Clear Canvas: Remove all images from the canvas.
  • Save Canvas: Save your current canvas layout (placeholder functionality).

Keyboard Shortcuts

  • Delete: Remove selected image from canvas.
  • H: Flip selected image horizontally (left-right mirror).
  • V: Flip selected image vertically (up-down mirror).
